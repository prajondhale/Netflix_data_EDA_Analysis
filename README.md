# 📺 Netflix Movies & TV Shows Exploratory Data Analysis (EDA)

This project demonstrates an end-to-end **Exploratory Data Analysis (EDA)** workflow, from **raw data cleaning** to **feature engineering** and **business insight generation** using Python.

---

# 📌 Project Overview

This project analyzes the **Netflix Movies & TV Shows** dataset using Python to understand the structure and trends of Netflix's content library.

The analysis includes:

- Data Cleaning
- Feature Engineering
- Exploratory Data Analysis (EDA)
- Business Insight Generation

The objective is to transform raw data into meaningful insights through systematic analysis and visualization.

---

# 🎯 Objectives

- Understand the structure of the Netflix dataset.
- Clean and preprocess the data.
- Handle missing and inconsistent values.
- Perform feature engineering.
- Explore trends using visualizations.
- Generate meaningful business insights.

---

# 🧹 Data Cleaning

The following preprocessing steps were performed:

- ✔ Checked data types
- ✔ Handled missing values
- ✔ Checked duplicate records
- ✔ Fixed inconsistent values
- ✔ Standardized categorical values
- ✔ Removed unnecessary columns

---

# ⚙️ Feature Engineering

To improve the quality of analysis:

- Extracted **Year**, **Month**, and **Day** from the date column.
- Split the **Duration** column into:
  - `duration_value`
  - `duration_unit`

This enabled separate analysis of **movie durations (minutes)** and **TV show seasons**.

---

# 📊 Business Questions Explored

1. What percentage of Netflix content consists of Movies and TV Shows?
2. Which maturity rating is the most common?
3. Which countries contribute the highest number of titles?
4. How has Netflix's content library grown over the years?
5. How have Movies and TV Shows changed over time?
6. Which genres dominate Netflix?
7. What is the distribution of movie durations?

---

# 🔍 Key Insights

- Movies dominate Netflix's content library compared to TV Shows.
- **TV-MA** is the most common maturity rating.
- The **United States** contributes the highest number of Netflix titles.
- Netflix experienced significant content growth after **2015**.
- **International Movies** and **Dramas** are among the most common genres.
- Most movies have a duration between **90–110 minutes**, indicating that medium-length movies dominate the catalog.
